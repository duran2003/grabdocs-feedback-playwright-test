# GrabDocs Feedback Automated Test

This repository contains an automated end-to-end test for the GrabDocs Feedback feature using Playwright.

## Files Included
- `feedback.test.js` – Playwright test that submits feedback through the GrabDocs UI
- `loginHelper.js` – Helper file used for authentication with a saved session

## Tool Used
- Playwright (JavaScript)

This test demonstrates black-box end-to-end testing of a web application feature.
